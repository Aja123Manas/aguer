What's the Difference?


HTML

Hypertext Markup Language

Create the structure

-Controls the layout of the content

-Produces structure for the web page design

-The facts about designs of every web page

CSS

Connecting Style Sheet

Stylize the website

-Applies styles to the webpage content

-Targets various screen sizes to make webpages responsive

-Primarly handles the "look good" on a webpage

Javascript

Increase Interactivity

-Adds interactivity in a webpage

-Handles complex functions and failures

-Programmatic codes which enhance functionality.
